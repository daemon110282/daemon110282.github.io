# PROMPT: Уточняющие вопросы по задаче

Сформируй список уточняющих вопросов по задаче:

---
{описание задачи}
---

Разбей вопросы на категории:

- бизнес‑требования
- пользовательские сценарии
- данные
- интеграции
- ограничения
- риски
- нефункциональные требования

Формат вопросов:

- нумерованный список
- перед номером вопроса добавлять символ markdown задачи [ ] для возможности отметить вопросы как отвеченные
