# PROMPT: Use Case

Сформируй Use Case для функции:

---

{описание функции}
---

Структура:

- Название
- Акторы
- Цель
- Триггер для старта Use Case
- Предусловия (Preconditions)
- Постусловия (Postconditions)
- Итоговое состояние
- Основной сценарий
- Альтернативные сценарии

Формат:

{ИД Use case} {Название Use Case}

| | |
|---|---|
|**Акторы**|{акторы}|
|**Цель**|{цель}|
|**Триггер для старта Use Case**|{триггер для старта Use Case}|
|**Предусловия**|{предусловия}|
|**Постусловия**|{постусловия}|
|**Итоговое состояние**|{итоговое состояние}|
|**Ссылка на User Story**||
|**Ссылки на другие Use Case**||

Шаги:

| № | Актор | Шаги основного сценария | Шаги альтернативного сценария | Ограничения |
|---|---|---|---|---|
| 1 | {актор} | 1.{шаг 1}<br>2. {шаг 2}<br>... | 1.{альтернативный шаг 1}<br>2. {альтернативный шаг 2}<br>... | - {ограничение 1} |
