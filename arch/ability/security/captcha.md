# Captcha

## Зачем

Captcha (Completely Automated Public Turing test to tell Computers and Humans Apart) — это тип задачи, которая используется для определения, является ли пользователь человеком или автоматизированной программой (ботом). Captcha часто используется на веб-сайтах для предотвращения автоматических регистраций, спама и других видов злоупотреблений.

Существует несколько типов Captcha, включая:

- Текстовые Captcha: Пользователю предлагается ввести текст, отображаемый на изображении с искажениями.
- Графические Captcha: Пользователю предлагается выбрать определенные изображения из набора.
- Invisible Captcha: Работает в фоновом режиме, анализируя поведение пользователя без необходимости взаимодействия.
- Audio Captcha: Предлагает пользователю прослушать аудиозапись и ввести услышанный текст, что полезно для людей с нарушениями зрения.
- Mathematical Captcha: Пользователю предлагается решить простое математическое уравнение, например, "2 + 3 = ?".

## Технологии

- reCAPTCHA: Сервис от Google, который предлагает различные формы проверки, включая "Я не робот" чекбоксы и анализ поведения пользователя.
- [Yandex SmartCaptcha](https://yandex.cloud/ru/services/smartcaptcha?utm_referrer=https%3A%2F%2Fwww.google.com%2F): Сервис от Яндекса, который использует машинное обучение для определения, является ли пользователь человеком.
	- [пример](https://habr.com/ru/articles/871848/)
- hCaptcha: Альтернативный сервис Captcha, который фокусируется на конфиденциальности пользователей и предлагает вознаграждения за участие в решении задач.

On premise:

- OpenCaptcha: Открытое решение для внедрения Captcha на собственных серверах.
- Captcha.NET: Библиотека для создания Captcha в .NET приложениях.
