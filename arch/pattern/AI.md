## Зачем

Технология, позволяющая машинам выполнять задачи, которые требуют человеческого интеллекта:

- распознавать речь
- анализировать изображения
- писать тексты
- принимать решения

Библиотека промтов для различных задач в области анализа требований, проектирования систем и взаимодействия с AI‑агентами.

- [ ] Использовать VS Code Snippets

- [ ] Библиотека промтов:
