# Apache Spark

## Зачем

Фреймворк с открытым исходным кодом для __параллельной обработки и анализа__ __слабоструктурированных данных__ в оперативной памяти.

Используется в архитектурном стиле [BigData](../../arch/style/bigdata.md).
